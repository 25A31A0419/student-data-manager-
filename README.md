 Student Data Manager in C

A simple menu-driven C project for storing and managing student records using structures, arrays, and functions.

This project allows users to:

- Add student details
- Display all student records
- Search student by ID

It is a beginner-friendly project for learning basic C programming concepts.

---

Features

- Add new student records
- View all saved student data
- Search student details using ID
- Easy menu-driven interface
- Uses "struct" for organized data storage

---

Technologies Used

- C Programming Language
- Standard C Library ("stdio.h", "string.h")

How to Run

Compile

gcc main.c -o student_manager

Run

./student_manager
